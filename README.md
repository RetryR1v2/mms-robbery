# mms-robbery

- Robbery Script

# Features
 
- Rob General Stores
- Rob Weapon Stored
- Rob Banks
- You need Lockpicks and Dynamite
- It will alert the LAW with GPS Route and Announcement
- you can use item and / or money as Reward
